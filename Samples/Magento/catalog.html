Indexed Database API Demonstration

This page demonstrates the utility of the Indexed Database API as a client-based data store. Here, any of the items listed below may be added to the shopping cart (via the Add to Cart links); these items are persisted entirely on the client and reconstructed on the shopping cart page. Note that this demonstration contains no server-side logic.

Note that most of the functionality on this page (e.g. filters, searching) have been deliberately disabled. This is designed to be a (very) simple demonstration of the Indexed Database API and not a full e-commerce simulation. The relevent code associated with these pages are located in the catalog.js and cart.js files.

This page (along with the shopping cart page) is derived from the Magento Community E-Commerce platform, which is available under an OSL 3.0 license. As such, this entire demonsration is available under the same.

Visit the project homepage to learn more about the Indexed Database API!
